Требования к функционалу библиотеки консольного взаимодействия с пользователем
1: Библиотека должна запускаться ( То есть не выдавать ошибки, сразу выводить текст чтобы пользовать не думал что все залагало )
2: Пользователь должен понимать что именно он ввел ( Пользователь должен видеть свой ввод )
3: Если пользователь ввел не то, что надо программе - сообщать и делать новый запрос на ввод ( Если пользователя попросили ввести кол-во, то он должен ввести цыфру, если строку - то строку, если он ввел не то, что требовалось, то обязательно сообщать ему об этом и снова просить чтобы он ввел )
4: Приглашение к вводу ( Пользователю должно выводиться что именно он должен ввести  )
5: Не использовать проф слова ( Не использователь слова из профессионального лексикона, чтобы он понимал что от него требуется )
